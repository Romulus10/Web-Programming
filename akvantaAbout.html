AKVANTA

The Game Wiki GitHub
Akvanta is a simple game written to run as an mobile web app. The game itself currently draws its inspiration from rougelike games that used ASCII-based graphics, displaying objects on the game maps through regular letters and characters. The game is located here. I would recommend reading the wiki's help page before playing.

Technical Details

The UI for Akvanta was written by hand in HTML and CSS- and was my introduction to large coding projects in the two languages. The game core itself is written in JavaScript, which I had not worked in extensively before taking on this project. As a result, I'm sure I made a number of efficiency and efficacy mistakes as well as general breaches of convention. Feel free to fork the project on GitHub and file pull requests!

The wiki for the game, which includes gameplay instructions and code documentation, is located on GitHub. Or now, play a text-based RPG adventure still under development!
